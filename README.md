El presente proyecto es un repaso por los principio físicos y teóricos del pulsioxímetro, recapitulando así lo aprendido y poniéndolo en práctica.

A la par, tiene el propósito de aprender a usar herramientas como lo son GitHub, Git y Micropython, esto a través de un prototipo de pulsioxímetro que sea capaz de medir la saturación de oxigeno en sangre. sirviendo como piedra angular para futuros proyectos.



Materiales:

ESP32



Recursos:

Se utilizo los drivers de la libreria MAX30102 del autor n-elias, cuyo contenido completo se encuentra en la siguiente liga:
